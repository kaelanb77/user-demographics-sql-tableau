User Demographics Analysis (SQL + Tableau)

This project demonstrates a complete data workflow using MySQL and Tableau.

Tools Used
MySQL (data creation and querying)
Tableau (data visualization)
Project Overview

I created a relational database to store user data, including name, age, and city. Using SQL, I performed analysis to understand user distribution and key metrics.
